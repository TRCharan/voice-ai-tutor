# Voice AI Language Tutor 🎙️🌍

A real-time, voice-enabled language learning application that acts as an interactive bilingual tutor. Built to handle the messy reality of Speech-to-Text (STT) audio, this app listens to users, decrypts phonetic microphone glitches, and responds with seamless, native-accented Text-to-Speech (TTS).

## ✨ Key Features

* **Smart Language Routing:** The LLM categorizes user intent on the fly. If you speak English, it acts as a translation dictionary. If you attempt the target language, it switches to a strict grading mode.
* **Bilingual Audio Engine:** Solves the "Browser Accent Bug." The React frontend dynamically chains English TTS and native Target Language TTS, ensuring the AI's instructions are clear and the translations sound native.
* **Phonetic Decryption:** Built-in safeguards against STT hallucinations. The backend sounds out glitchy audio (including cross-script hallucinations like Japanese or Hindi text) to deduce the user's true intent before grading.
* **Silence Filtering:** Custom backend logic intercepts and deletes "ghost phrases" (e.g., "Thanks for watching") generated by Whisper during dead air.
* **Contextual Memory:** Integrates RAG (Retrieval-Augmented Generation) to remember past context while strictly separating factual memory from language grading.

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Python, FastAPI
* **AI & LLM:** Llama 3.1 (8B) via Groq API
* **Speech-to-Text:** Whisper-large-v3 via Groq API
* **Memory / Database:** Vector Database (ChromaDB)

## 🚀 Getting Started

### Prerequisites
* Node.js and npm
* Python 3.9+
* A [Groq API Key](https://console.groq.com/keys)

### 1. Clone the Repository
```bash
git clone [https://github.com/YOUR-USERNAME/voice-ai-tutor.git](https://github.com/YOUR-USERNAME/voice-ai-tutor.git)
cd voice-ai-tutor

2. Backend Setup (FastAPI)
Navigate to the backend directory, create a virtual environment, and install dependencies.

Bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
pip install -r requirements.txt
Create a .env file in the backend directory and add your API keys:

Code snippet
GROQ_API_KEY=your_groq_api_key_here
Start the backend server:

Bash
uvicorn app.main:app --reload
3. Frontend Setup (React)
Open a new terminal window, navigate to the frontend directory, and start the development server.

Bash
cd frontend
npm install
npm run dev
🧠 Architecture Notes
Developing real-time Voice AI requires significant "duct tape" to handle raw audio data. This project specifically implements a Category-Based State Machine in the LLM prompt to prevent "helpful chatbot" behavior, forcing the AI to act as a strict software component that grades syntax rather than policing the user's factual statements.
